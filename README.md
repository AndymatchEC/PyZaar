## 📋 What is PyZaar?

PyZaar is a simple app designed to help you with Hypixel Skyblock’s Bazaar. It shows you important price information and helps you decide when to buy and sell items. The app uses Python and PySide 6 to offer a clear visual guide that anyone can use.

It’s made for players who want a better way to track item prices and make smart flipping decisions without having to search through multiple websites or tools.

---

## 🔍 Key Features

- Shows live item prices from Hypixel Skyblock Bazaar
- Visual charts for price trends and fluctuations
- Helps identify good flipping opportunities
- Simple and clear interface for all skill levels
- Works on Windows without extra setup
- Updates data automatically to keep prices current

## 💻 System Requirements

PyZaar runs on Windows computers. Make sure your system matches these:

## ⚙️ Troubleshooting

If you run into issues, try these steps:

- **App Won’t Open:**  
  Restart your computer and try again. Make sure you ran the installer as an administrator.

- **Prices Not Updating:**  
  Check that your internet connection works. Close and reopen PyZaar.

- **Installation Fails:**  
  Make sure you have enough disk space and that no other antivirus software blocks the installer.

- **App Crashes:**  
  Try reinstalling from the download page.

You can also look at the issues tab on the PyZaar GitHub page for help from the community.
